Welcome to SNAPPY's documentation!
===================================

**SNAPPY** is a Python program used to assign Y-chromosome phylogeny-informed haplogroups using dense genotype data.
SNAPPY's reference library is pulled from `Open Food Facts database <https://isogg.org/>`.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
